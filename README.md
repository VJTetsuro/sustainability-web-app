# Sustainability Web Application

## Overview
This project is a full-stack web application developed as part of a university group assignment. The application presents selected UN Sustainable Development Goals and allows users to sign up for newsletters.

The system dynamically loads content using JavaScript and stores user data via a Node.js and Express backend.

---

## Features
- Dynamic content loading using JavaScript (Fetch API + JSON)
- Multi-page website (Home, Goals, Team, Signup)
- User signup form with validation
- Backend server using Node.js and Express
- Data storage in JSON format
- Confirmation response after form submission

---

## Technologies Used
- HTML5 (semantic markup)
- CSS3 (external styling)
- JavaScript (DOM manipulation, Fetch API)
- Node.js
- Express.js
- JSON (data storage)
